Book REST Webservice Description.

Service Name Method End Point Description Important Note
All Book GET http://localhost:9090/service/book/all This End point provides all data. NOTE : May give no output if there is no data.
Get Book By Id GET http://localhost:9090/service/book/{id} This End point provides single record data according to {id} given in url. NOTE : The URL only display record for id 7. only
Create new Book Record POST http://localhost:9090/service/book Example : { "name" : "Mastering Apache Spark" } NOTE : Should use a REST client. !
Update Existing Book Record PUT http://localhost:9090/service/book Example : { "id" : 1, "name" : "Mastering Apache Spark_Update" } NOTE : Should use a REST client. !
Delete Existing Book By Id DELETE http://localhost:9090/service/book/{id} Example : { "id" : 1, "name" : "Mastering Apache Spark_Update" } NOTE : Should use a REST client. !
